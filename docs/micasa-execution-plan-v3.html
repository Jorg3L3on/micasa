No password-reset email path (out of scope). A forgotten password currently means direct DB intervention — and there is no admin UI. Assessment: this is the single biggest operational gap and compounds with the recovery hole. Minimal /admin: user lookup + temporary password set (communicate out-of-band).
Medium Blocker · 0/5 ↑ Ops + Auth recovery
