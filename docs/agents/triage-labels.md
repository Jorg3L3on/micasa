# Triage labels

| Canonical role    | GitHub label        | Meaning |
| ----------------- | ------------------- | ------- |
| `needs-triage`    | `needs-triage`      | Needs evaluation |
| `needs-info`      | `needs-info`        | Waiting on reporter |
| `ready-for-agent` | `ready-for-agent`   | Ready for AFK agent |
| `ready-for-human` | `ready-for-human`   | Needs human |
| `wontfix`         | `wontfix`           | Will not fix |

**Change type:** `type:feature`, `type:bug`, `type:chore`, `type:docs`

**Status (optional):** `status:in-progress`

Create labels: `bash scripts/create-github-labels.sh`
